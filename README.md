# 🎵 Spotify Clone

A Spotify-inspired music player built using **HTML**, **CSS**, and **JavaScript**. This project was created to practice front-end web development and improve JavaScript skills.

## 🚀 Features

- 🎵 Play and pause songs
- ⏭️ Next and previous track
- 📃 Playlist display
- 📱 Responsive design
- 🎨 Clean and modern user interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

SpotifyClone/
